switch statement: It is used to compare a value with various cases and execute the corresponding block of code. 
    
        switch (variable) {{"{"}}
            case value1:
              // code to be executed if variable matches value1
              break;
            case value2:
              // code to be executed if variable matches value2
              break;
            ...
            default:
              // code to be executed if variable does not match any value
              break;
          {{"}"}}
Example. 
    
        int day = 3;
switch(day)
{{"{"}}
   case 1:
      std::cout << "Monday";
      break;
   case 2:
      std::cout << "Tuesday";
      break;
   case 3:
      std::cout << "Wednesday";
      break;
   default:
      std::cout << "Invalid day";
{{"}"}}
Note: The break statement is used to exit the switch statement after a case has been executed, otherwise, it will continue executing the following cases.
