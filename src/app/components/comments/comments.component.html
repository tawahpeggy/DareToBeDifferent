Comments

Comments can be used to explain code, and to make it more readable. It can also be used to prevent execution when testing alternative code.

We have two types of comments, the single-lined and the multi-lined Comments.

Single line Comments

Single-line comments start with two forward slashes ( // ).

Any text between // and the end of the line is ignored by the compiler that is, will not be executed.

Let's have an example ; 
    
        // This is a comment
cout << "Hello World!";
Also,
cout << "Hello World!"; // This is a comment
Multiple line Comments

They are used to describe code in more than one line or hide large blocks of text or code from the compiler. All the text between /* and */ is ignored by the compiler.
