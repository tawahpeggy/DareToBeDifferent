Setting up a c++ Environment in Windows

In setting c++ environment in windows, you will follow the following steps.

Step 1

Download and install a c++ compiler. One popular visual studio, which is a comprehensive IDE that supports c++. You can download it for free from Microsoft’s website.

Step 2

Once you have installed visual studio, you can create a new C++ project. Go to file >new >project and then select a C++ project template. You can choose from several options, such as console applications, and universal Windows platform [UWP] applications.

Step 3

After you have created a project, you can start writing your code. Open the main.cpp file, which contains the starting point of your program. Here’s an example of a ‘’Hello, World’’ program.
