C++ Pointers

C++, pointers are variables that store the memory addresses of other variables.

A pointer variable points to a data type (like int or string) of the same type, and is created with the * operator. The address of the variable you are working with is assigned to the pointer.

Look at the code snippet below: 
    
    string drinks = "Fanta";  // A drinks variable of type string
    string* ptr = &drinks;    // A pointer variable, with the name ptr, that stores the address of drinks
    // Output the value of drinks (Fanta)

    cout << drinks << "\n";
    
    
        
// Output the memory address of drinks (0x6dfed4)
    cout << &drinks << "\n";
    

    
        // Output the memory address of drinks with the pointer (0x6dfed4)
         cout << ptr << "\n";
Explanation of above code:

Create a pointer variable with the name ptr, that points to a string variable, by using the asterisk sign * (string* ptr) . Note that the type of the pointer has to match the type of the variable you are working with.

Use the & operator to store the memory address of the variable called drinks, and assign it to the pointer. Now, ptr holds the value of the drinks' memory address.

Ways of declaring pointers
There are three ways of declaring pointers, but the first is the most preferred. They are:
  • string* mystring; // Preferred
  • string *mystring;
  • string * mystring;

Note: The * operator is used after the data type to declare pointers.

Assigning Addresses to Pointers
    
        int* pointVar, var;
        var = 5;
// assign address of var to pointVar pointers 
    
        pointVar = &var;
Here, 5 is assigned to the variable var. And, the address of var is assigned to the pointVar pointer with the code 
    
        pointVar = &var.
Get the Value from the Addres Using Pointers. 
    
        int* pointVar, var; var = 5; // assign address of var to pointVar.
        pointVar = &var;
access value pointed by pointVar 
    
        cout << *pointVar << endl;   // Output: 5

n the above code, the address of var is assigned to pointVar. We have used the *pointVar to get the value stored in that address.

When * is used with pointers, it's called the dereference operator. It operates on a pointer and gives the value pointed by the address stored in the pointer. That is, *pointVar = var

How C++ Pointers work.
Find below a code snippet to show the working of pointers in C++ 
    
        
#include <iostream>
using namespace std;
int main() {{"{"}}
    int var = 5;

    // declare pointer variable
    int* pointVar;

    // store address of var
    pointVar = &var;

    // print value of var
    cout << "var = " << var << endl;

    // print address of var
    cout << "Address of var (&var) = " << &var << endl
         << endl;

    // print pointer pointVar
    cout << "pointVar = " << pointVar << endl;

    // print the content of the address pointVar points to
    cout << "Content of the address pointed to by pointVar (*pointVar) = " << *pointVar << endl;
    
    return 0;
{{"}"}}
Output var = 5 Address of var (&var) = 0x61ff08 pointVar = 0x61ff08 Content of the address pointed to by pointVar (*pointVar) = 5
Modifying the values of pointers.
    
        string food ="Pizza";
string* ptr = &food;
// Output the value of food (Pizza) 
    
        cout << food << "\n";
// Output the memory address of food (0x6dfed4) 
    
        cout << &food << "\n";
// Access the memory address of food and output its value (Pizza) 
    
        cout << *ptr << "\n";

        *ptr = "Hamburger";
Output the new value of the pointer (Hamburger) 
    
        cout << *ptr << "\n";
