C++ Vectors

In C++, a vector is a dynamic container that allows you to store and manipulate elements in an efficient manner. Here are some notes on vectors in C++:

Creating a vector

You can create a vector by declaring it and specifying the type of elements it will hold. For example, you can create a vector of integers as follows: 

        
            #include <vector>
            std::vector<int> my_vector;
Adding elements

You can add elements to a vector using the push_back method. For example:

    
            my_vector.push_back(1);
            my_vector.push_back(2);
            my_vector.push_back(3);
Accessing elements

Accessing elements You can access elements in a vector using the square brackets operator.

For example: 
        
            std::cout << my_vector[0]; // Prints 1
            std::cout << my_vector[1]; // Prints 2
Vector size

You can get the size of a vector by calling its size() method.

For example: 
    
        std::cout << my_vector.size(); // Prints 3
Vector iteration

You can iterate over a vector using a range-based for loop.

For example: 
    
        for (int i : my_vector) {{"{"}}
            std::cout << i << std::endl;

Removing elements

You can remove elements from a vector by calling its erase method.

For example: 
    
        my_vector.erase(my_vector.begin() + 1); // Removes element at index 1
Sorting elements

You can sort the elements in a vector using the sort function in the <algorithm> library.

For example: 
    
        #include <algorithm>

        std::sort(my_vector.begin(), my_vector.end());

These are just a few of the most commonly used functions and operations associated with vectors in C++. There are many other methods you can use to manipulate and access vector elements, so it's a good idea to consult the documentation to learn more.
