Inheritance

Inheritance is a feature in C++ that allows a class to inherit the properties and behaviors of another class. The class that is being inherited from in called the base class, while the class that is inheriting from the base class is called the derived class

To create a derived class in C++ use the syntax below: 
    
        class DerivedClass: accessSpecifier

The access specifier can be either public , protected , or private and determines how the derived class can access the members of the base class.

In public inheritance, the public members of the base class become public members of the derived class, the protected members of the base class become protected members of the derived class, and the private members of the base class are not accessible by the derived class.

In Protected inheritance, the public and protected members of the base class become protected members of the derived class, and the private members of the base class are not accessible by the derived class.

In Private Inheritance, the protected and public members of the class become private members of the derived class, and the protected members of the base class are not accessible by the derived class.

In C++, there are three types of inheritance:

1. Single inheritance: In single inheritance, a derived class inherits the properties of only one base class. The derived class inherits the data members and member functions of the base class.

2. Multiple inheritance: In multiple inheritance, a derived class can inherit the properties of more than one base class. This means that the derived class has multiple base classes and it can access the data members and member functions of all of them. 3. Multilevel inheritance: In multilevel inheritance, a derived class is created from another derived class. This means that the base class for the derived class is itself a derived class. In this type of inheritance, the properties of the first derived class are inherited by the second derived class, and so on.
